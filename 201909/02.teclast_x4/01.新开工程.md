## 1. 新建工程

```shell
# hp-4.19/.repo/manifests$
git checkout -b edu_tec_x4
git push --set-upstream origin edu_tec_x4
```

## 2. 同步工程

方法一：

```shell
# hp-4.19$
repo init -u ssh://caobinxin@192.168.1.112:29418/x86manifest -b edu_tec_x4
repo sync -c -j4
```



方法二：

用该方法切换工程

```shell
repo forall -c git reset --hard
repo init -u ssh://caobinxin@192.168.1.112:29418/x86manifest -b edu_tec_x4
repo sync
```
