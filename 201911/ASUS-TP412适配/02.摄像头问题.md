# 1. 问题描述

录像时,提示 无法连接到相机,

# 2. 问题分析：

## 1. logcat

先看log 中，那个报错：

```shell
E/CameraSource( 2007): Video dimension (1280x720) is unsupported
```

## 2. 相机的参数

当前这个参数是从 ubuntu 下去拿的。

从中我们发现，我们的相机并不支持 1280x720 格式的， 所以，我们只需要将我们的这种格式删除即可。
